# 학습 완료 상태로 변경하는 방법

각 괄호(`[ ]`) 안에 영문 엑스(`x`)를 입력하고 저장하면 `완료 상태 [x]`로 변경됩니다. 아래 예시코드를 참고하세요.

```md
- [x]  JavaScript란? (2분 41초)
- [x]  최고의 교과서 (9분 8초)
- [x]  워밍 업! - Console 패널 (2분 17초)
- [x]  코멘트 / 디버깅 (4분 22초)
- [ ]  선언 / 할당 (14분 22초)
- [ ]  Start! 인터랙션 (5분 31초)
```
